Client

Browser dell'utente, accede via HTTPS.

Vercel — Frontend

Astro SSR con TailwindCSS, middleware di autenticazione, adapter Vercel.

Railway — Backend

FastAPI + Uvicorn su Python 3.11, containerizzato via Docker, esposto su $PORT.

Supabase

Auth (JWKS JWT), PostgreSQL con pooler su :6543, 8 tabelle applicative.

External APIs

Anthropic Claude per AI search/evaluation, YouTube tramite yt-dlp per i transcript.
